# 🚀 Python-Pro: Space Mission Analytics Dashboard

A data visualization and analytics dashboard built in Python that explores more than 60 years of global space mission history (1957–2022).

The project analyzes launch trends, mission success rates, space organizations, launch countries, mission costs, and historical space exploration eras through interactive visualizations.

---

## 📊 Project Overview

This dashboard provides insights into:

- Total space missions conducted worldwide
- Mission success and failure rates
- Launch activity across different historical eras
- Top space organizations by mission count
- Country-wise mission statistics
- Cost analysis of space programs
- Most expensive missions
- Active rocket statistics

The goal is to transform raw mission datasets into meaningful visual insights for students, researchers, and space enthusiasts.
